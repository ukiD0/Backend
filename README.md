### Try do teastapi 

### For install and run
1. Install
   - ```install.bat``` (for Windows)
   - ```bash install.sh``` (for Linux)
2. Run
   - ```run.bat``` (for Windows)
   - ```bash run.sh``` (for Linux)
3. Create testdata
   - open new terminal (cmd)
   - activate env (if not activated)
     - for activate env .\env\Scripts\activate.bat
   - ```python testdata.py```
